# DuressKeyboardLite

Safe and fast keyboard with the DuressPassword feature supported on phones where custom keyboards can work in password input fields.

Duress Password — a phone data reset code, configurable in this app, that you enter via this keyboard on the lock screen instead of the password and send by pressing the Enter arrow (⏎) if unlocking under coercion.

This app — a simplified version of DuressKeyboard with reduced functionality, but better compatibility with new Android versions.
