# DuressKeyboardLite

Safe and fast keyboard app with the DuressPassword feature, supported on phones where custom keyboards can work in password input fields.

Duress Password — a phone data reset code configurable in this app that you enter via this keyboard in the screen lock password input field and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone.

This app — a simplified version of DuressKeyboard with reduced functionality, but better compatibility with new Android versions.
