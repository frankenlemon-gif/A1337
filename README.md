# DuressKeyboardLite

Safe and fast keyboard with the DuressPassword feature.

Duress Password — a phone data reset code configurable in this app. Enter it via this keyboard instead of a lock screen password and press Enter (⏎) if unlocking under coercion.

For this feature to work, the keyboard must be set as the default on a permanent basis, and your phone must support its use in password input fields.

This app — a simplified version of DuressKeyboard with reduced functionality, but improved new Android compatibility.
