# DuressKeyboardLite

Safe and fast keyboard app with DuressPassword feature, supported on phones where custom keyboards can work in password input fields.

Duress Password — a data reset code configurable in this app that you enter via this keyboard in the password input field on the lock screen and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone.

This app — simplified version of DuressKeyboard with reduced set of capabilities, but with better compatibility with new Android versions.
