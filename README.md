# DuressKeyboardLite

Safe and fast keyboard with the DuressPassword feature.

Duress Password — a phone data reset code, configurable in this app, that you enter via this keyboard instead of your ordinary password on the lock screen, and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone.

For this feature to work, you must set this keyboard as default and your phone must not be Chinese/Realme/Oppo in order to support its use in password input fields.

This app is a simplified version of DuressKeyboard with reduced functionality, but improved compatibility for newer Android versions.
