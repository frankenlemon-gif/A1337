Keyboard with Duress Password feature.

Duress Password — a data reset code configurable in this app you enter via this keyboard in the password input field on the lock screen and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone.

This feature is available in phones where third-party keyboards can work on the lock screen, which means all Android phones except some Chinese ROMs.

This app also has these features: 
1. Set Your Limit of incorrect password entry attempts containing more than 4 characters to wipe data (main menu -> Auto-Wipe settings).
2. Set Base Limit of incorrect password entry attempts containing more than 4 characters to wipe data to 1 and increase it up to 5 while the keyboard is used — to wipe data on any incorrect password entry containing more than 4 characters when an attacker bypasses the keyboard to bypass DuressPassword, and not wipe when you use it. (main menu -> Dead Hand mode).

This keyboard doesn't collect your input and doesn't have internet access.

This app is a simplified version of DuressKeyboard app, it has higher target SDK, which means easier installation, but fewer features.
