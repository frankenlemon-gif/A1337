Keyboard with Duress Password feature.

Duress Password — a data reset code configurable in this app. You enter it via this keyboard in the password input field on the lock screen and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone.

This feature is available in phones where third-party keyboards can work on the lock screen, which means all Android phones except some Chinese ROMs.

This app also has these features: 
1. Set Your Limit of failed password attempts to wipe data (main menu -> Auto-Wipe settings).
2. Set Base Limit of failed password attempts to wipe data to 1 and increase it up to 5 while keyboard is used — to wipe data when attacker bypasses keyboard to bypass DuressPassword and not wipe when you use it. (main menu -> Dead Hand mode).

This keyboard doesn't collect your input and doesn't have internet access.

This app is a simplified version of DuressKeyboard app, and it has higher target SDK, which means easier installation, but fewer features.
