Keyboard with Duress Password feature, without internet access and data collection; simplified version of DuressKeyboard with limited functionality but easier installation.

Duress Password — a data reset code configurable in this app that you enter via this keyboard in the password input field on the lock screen and send by pressing the Enter arrow (⏎) when someone forces you to unlock the phone. Supported only on non-Chinese phones where custom keyboards can be launched in password input fields.
